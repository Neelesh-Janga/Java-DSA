# Linked List Operations

## I. Singly Linked List

### 1. Create - `create(int... varArgs)`

- **Time Complexity:** O(n), where n is the number of elements in the linked list.
- **Space Complexity:** O(1)

### 2 . Display - `display(SinglyLinkedListNode head)`

- **Time Complexity:** O(n), where n is the number of elements in the linked list.
- **Space Complexity:** O(1)

### 3. Insert - `insert(SinglyLinkedListNode head, int number, int index)`

- **Time Complexity:** O(n), where n is the number of elements in the linked list.
- **Space Complexity:** O(1)

### 4. Find - `find(SinglyLinkedListNode head, int number)`

- **Time Complexity:** O(n), where n is the number of elements in the linked list.
- **Space Complexity:** O(1)

### 5. Find Length of List - `findLength(SinglyLinkedListNode head)`

- **Time Complexity:** O(n), where n is the number of elements in the linked list.
- **Space Complexity:** O(1)

### 6. Concatenate Two Lists - `concat(SinglyLinkedListNode list1, SinglyLinkedListNode list2)`

- **Time Complexity:** O(n), where n is the total number of elements in list1 and list2.
- **Space Complexity:** O(1)

### 7. Reverse - `reverse(SinglyLinkedListNode head)`

- **Time Complexity:** O(n), where n is the number of elements in the linked list.
- **Space Complexity:** O(1)

### 8. Reverse Using Recursion - `reverseUsingRecursion(SinglyLinkedListNode head)`

- **Time Complexity:** O(n), where n is the number of elements in the linked list.
- **Space Complexity:** O(n), due to the recursion stack.

### 9. Loop Detection - `isLooped(SinglyLinkedListNode head)`

- **Time Complexity:** O(n), where n is the number of elements in the linked list.
- **Space Complexity:** O(1)

### 10. Delete - `delete(SinglyLinkedListNode head, int number)`

- **Time Complexity:** O(n), where n is the number of elements in the linked list.
- **Space Complexity:** O(1)

### 11. Find Middle Number - `findMiddleNumber(SinglyLinkedListNode head)`

- **Time Complexity:** O(n), where n is the number of elements in the linked list.
- **Space Complexity:** O(1)

### 12. Palindrome Check - `isPalindrome(SinglyLinkedListNode head)`

- **Time Complexity:** O(n), where n is the number of elements in the linked list.
- **Space Complexity:** O(n), due to the space required for the copied list. (Optimization needed to get O(1))

### 13. Get a Copy - `copy(SinglyLinkedListNode head)`

- **Time Complexity:** O(n), where n is the number of elements in the linked list.
- **Space Complexity:** O(n), due to the space required for the copied list.

### 14. Get Intersection Node - `getIntersectionNode(SinglyLinkedListNode list1, SinglyLinkedListNode list2)`

- **Time Complexity:** O(n), where n is the number of elements in the linked list.
- **Space Complexity:** O(1)

### 15. Delete Entire List - `deleteEntireList(SinglyLinkedListNode head)`

- **Time Complexity:** O(n), where n is the number of elements in the linked list.
- **Space Complexity:** O(1)

### 16. Remove Loop - `removeLoop(SinglyLinkedListNode head)`

- **Time Complexity:** O(n), where n is the number of elements in the linked list.
- **Space Complexity:** O(1)

### 17. Merge Sorted Linked Lists - `mergeSortedLinkedLists(SinglyLinkedListNode list1, SinglyLinkedListNode list2, boolean ascending)`

- **Time Complexity:** O(m + n), where m and n are the lengths of list1 and list2, respectively.
- **Space Complexity:** O(1)

### 18. Get Union of Two Lists - `findUnion(SinglyLinkedListNode list1, SinglyLinkedListNode list2)`

- **Time Complexity:** O(m + n), where m and n are the lengths of list1 and list2, respectively.
- **Space Complexity:** O(m + n), where m and n are the lengths of list1 and list2, respectively.
